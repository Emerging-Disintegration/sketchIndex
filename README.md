# SketchIndex

SketchIndex is a reference image library and drawing practice toolkit for artists. Existing drawing reference tools lack visual quality and category depth — SketchIndex aims to close that gap.

## Goals

- Provide a browsable library of high-fidelity reference images organized by category (figures, body parts, objects, vehicles, etc.)
- Let artists filter references by viewing angle and lighting condition
- Offer timed drawing practice sessions (Study Mode)
- Give artists a personal space to save and organize reference collections (Studio Mode)
- Encourage consistent practice through daily drawing prompts
